⚙️ Configuração

Como gerar o token?

Nova API (developers.deriv.com) — tokens pat_xxx:

  1. Acesse developers.deriv.com e faça login
  2. Vá em Profile → API tokens
  3. Crie um token PAT com permissão Trade
  4. O token começa com pat_ — copie e cole aqui

API Legada (legacy-api.deriv.com) — tokens a1-xxx:

  1. Acesse legacy-api.deriv.com e faça login
  2. Crie um token com permissões Read e Trade
  3. O token começa com a1-

⚠️ Nunca compartilhe seu token com ninguém.
